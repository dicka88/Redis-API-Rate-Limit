# API RATE LIMIT
Example implementat API rate limit on Node JS js with Redis storage

## Install
```bash
npm install
```

## How to run
```bash
npm run start
```

## How to test[default]
```bash
Open localhost:3000 and refresh 10x to see api rate limit before 1 minutes
```

## Tech
- Node js
- Redis
- Redis js
- Bluebird
- Moment
